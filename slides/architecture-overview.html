CORE FEATURES · 1/2

核心功能 — AI & 运维

🤖

AI Agent 智能代理

Claude AI 实时接入全部集群、指标、日志，执行 kubectl / PromQL / LogQL 工具调用

  • 动态上下文注入 — 每次会话自动加载账号、集群、可观测性端点
  • MCP 工具协议 — 安全扫描、计费分析、浏览器自动化等可扩展能力
  • Skills 系统 — Git 发现、安装、热重载，支持 User/Tenant/Global 三级作用域
  • 流式 SSE 推送 — 实时展示推理过程和工具调用结果
🚀

渐进式部署 Argo Rollouts

集成 Argo Rollouts 的完整渐进式交付能力

  • Canary — 分步流量切换，自动化分析
  • Blue-Green — 零停机切换 + 即时回滚
  • 可视化拓扑 — Ingress → Service → Rollout 实时状态图
  • 一键 Promote / Rollback 操作
📈

多云可观测性 Unified Telemetry

跨 Provider 统一遥测路由，自动匹配运行时环境

  • 自建 LGTM 或 Grafana Cloud 两种模式
  • Grafana / Datadog / Dynatrace 告警 Webhook 接入
  • CloudWatch Anomaly Detection 异常检测
  • AI 自动关联 Metrics + Logs + Traces

WHY OPS K

项目优势 六大支柱

🤖

AI-Native SRE 智能运维

全量基础设施上下文注入，实时执行 kubectl / PromQL / LogQL 工具调用，流式输出推理过程，告警到根因分析闭环自动化。

Claude CLI Subprocess MCP Protocol SSE Streaming Skills Hot-reload
🛡

Zero Trust 纵深防御

WAF 边缘过滤 → ALB TLS 终止 → 多租户 RBAC 隔离 → 账号级读写权限 → Secrets Manager 零明文存储。敏感操作需 Approval Workflow 审批。

WAF v2 RBAC 3-Tier ESO → K8s Secrets Approval Workflow
🟢

High Availability 高可用架构

Multi-AZ VPC + Aurora 自动故障转移 + EKS 托管控制面 + Karpenter 节点自愈。Argo Rollouts Canary / Blue-Green 渐进式发布，零停机交付。

Multi-AZ Auto Failover Canary Rollout Blue-Green
📈

Full Observability 全栈可观测

LGTM 栈覆盖 Metrics / Logs / Traces 三大信号。支持自建或 Grafana Cloud 两种模式，AI Agent 按运行时自动路由查询，关联分析无需手动切面板。

PromQL LogQL TraceQL Grafana Cloud
⚡

Cost Efficient 成本优化

Graviton ARM64 节点成本降低 ~40%，Karpenter 按需伸缩精准匹配负载，Aurora Serverless v2 按用量计费。告别固定容量预留，弹性即省钱。

Graviton ARM64 Karpenter Serverless v2
🛠

IaC & GitOps 全自动化交付

Terraform 管理全部基础设施，ArgoCD 声明式部署应用。一条命令从零搭建完整环境，代码即基础设施，Git 即唯一真相源。

Terraform ArgoCD Helm One-click Deploy
